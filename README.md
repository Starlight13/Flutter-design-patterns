# design_patterns

A Flutter project demonstrating various design patterns.

## Included design patterns

- Creational 
    - Singleton
    - Builder
    - Abstract factory
    - Factory method
    - Prototype
- Structural
    - Flyweight
    - Proxy
    - Composite
    - Decorator
    - Adapter
- Behavioral
    - Strategy
    - Memento
    - Observer
    - Chain of responsibility
    - Mediator
